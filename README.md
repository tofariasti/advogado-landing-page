## ✨ Funcionalidades

### 📱 Design & UX
- ✅ Design premium dourado/preto transmitindo sofisticação e confiança

## 🚀 Como Executar Localmente

### Pré-requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Não requer Node.js ou dependências (HTML/CSS/JS puro)

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/tofariasti/advogado-landing-page.git
cd advogado-landing-page
```

2. Abra o arquivo `index.html` no navegador:
```bash
# Linux
xdg-open index.html

# macOS
open index.html

# Windows
start index.html
```

Ou use um servidor local (recomendado):
```bash
# Python 3
python -m http.server 8000

# Node.js (com npx http-server)
npx http-server

# PHP
php -S localhost:8000
```

3. Acesse no navegador: `http://localhost:8000`

## 🔧 Customização

Para adaptar este template para seu escritório:

1. **Identidade Visual**
   - Edite as variáveis CSS em `site/assets/css/style.css` (`:root`)
   - Substitua cores douradas pela paleta da sua marca
   - Ajuste tipografia se necessário

2. **Conteúdo**
   - Edite `site/index.html` com seus textos
   - Adicione logo e imagens reais
   - Atualize áreas de atuação conforme especialidades

3. **WhatsApp**
   - Substitua o número em `site/assets/js/main.js` (linha 235)
   - Personalize a mensagem do formulário

4. **Informações de Contato**
   - Atualize endereço, telefone, e-mail no footer
   - Adicione número OAB real
   - Configure redes sociais
